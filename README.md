# 易物 (Yi Wu)

B2B social platform connecting verified Chinese manufacturers with overseas designers and buyers.

## Stack
- Next.js 15 (App Router), TypeScript, Tailwind CSS
- Supabase (Auth + PostgreSQL + RLS)
- next-intl (EN/ZH/ES/IT)
- Vercel deployment

## Features
- World Wall social feed
- Invitation Hall (RFP system)
- In-app chat with DeepL translation
- Manufacturer audit verification
- Admin backend
- 4-language support
